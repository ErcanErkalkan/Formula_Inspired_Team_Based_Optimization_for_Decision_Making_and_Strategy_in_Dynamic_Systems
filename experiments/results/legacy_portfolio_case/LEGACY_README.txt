Legacy portfolio-case outputs
=============================

These files were generated by the original five-algorithm portfolio case-study
script. They are retained for traceability only.

Do not use these files for the ASOC manuscript portfolio tables. The active
ASOC portfolio pipeline is experiments/run_portfolio_asoc_suite.py and writes
asoc_portfolio_* outputs with the full MAIN_ALGORITHMS list:
FITO, DNSGA-II-A, DNSGA-II-B, KGB-DMOEA, MDDM-DMOEA, NSGA-II, PPS-DMOEA.
